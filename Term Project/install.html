Materialize Installation

STEP 1: Open Materialize web page on your web browser. The page is available in the following link: https://materializecss.com/



STEP 2: Click on the Get Started button and scroll down to the Download section. Download the standard version of Materialize



STEP 3: Check your Downloads folder and unzip the downloaded zip file



STEP 4: Open the unzipped Materialize folder and check if all the listed files below are present (aside from index.html)



STEP 5: Create your own index.html file in the folder and make sure it is located in the correct directory



STEP 6: Go back to the Getting Started page and find the HTML Setup section containing the following HTML codes



STEP 7: Copy and paste the code provided in the HTML Setup section to your index.html file. Start adding your HTML and Materialize codes in the highlighted space





Django Installation

STEP 1: There are 3 different ways to install Django and the following installation steps uses pip in virtual environment.
Once you are in the server's home directory, create a new directory for the django application (command: mkdir name) and go into directory (command: cd name)



STEP 2: In the django-apps directory, create a virtual environment and name it env (command: virtualenv env)



STEP 3: Activate the virtual environment with the command . env/bin/activate



STEP 4: To check if the virtual environment activated, check if the prefix is changed to (env) and the name will depend on what directory you are in



STEP 5: With the active environment, install the Django package using pip. This installation will allow us to create and run Django application. Use the command pip install django in your terminal to install



STEP 6: Once it is done installing, verify the Django installation by doing a version check with the command django-admin --version
