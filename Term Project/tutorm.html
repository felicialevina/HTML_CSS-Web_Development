Creating Your Page with Materialize

Tutorial
This tutorial will cover the following topics:
1. Images: Inserting your downloaded images onto page
2. Responsive Images: Your images don't have to be in a fixed location on your page. Using a "responsive-img" tag, images can resize responsively
3. Icons: Importing Materialize icons on page
4. Buttons: Materialize offers different designs to personalize your buttons from floating buttons, raised buttons and 'pulse' buttons
5. Creating 'Cards' to hold images and information. Cards are similair to text-boxes an act as container/backdrop to your images and information
6. Specify exact location of page's components using 'Grid'

The following explanations use a demo "Weekly Schedule" page as basis for explaining different feautures of Materialize

Images

To import a downloaded image from your files use tag with the saved name of the image.

Example: <img src="parallax1.jpg>

Responsive Images

To make your image resize responsively add class="responsive-img" to your image tag The following code will use a responsive image as the backdrop for your Weekly Schedule page:

Code: <img src="parallax1.jpg" class="responsive-img">

Result:

Icons

To use Materialize icons on your page use the: <i class="material-icons>" code with the name of the icon you'd like to import. The name of all the icons can be found on the Materialize site at https://materializecss.com/icons.html

Example using icon called "event_note": <i class="material-icons #ef9a9a-text" style="font-size: 150px">event_note</i>

Result: Highlighted is the icon on Demo Page:

Cards

'Cards' are very similair to textboxes in Materialize. They allow for a backdrop/container for your images and information

- Cards are made using <div class="card"> code

- To personalize card you can add more details to line of code: <div class="card blue-grey darken-1"> OR import an image into the card with the image tag

- To write text in card, use span class: <span class="card-title center-align">

Example on creating and personalizing card using an imported image and adding a title:

<div class="card">

<div class="card-content">

<img src="exam1.jpg">

</div>

<span class="card-title center-align">Got an exam? Set up your studying schedule!</span> </div>

Buttons

You can use buttons as an interactive component on your page using: <a class="btn">button</a>
Result:
button

You can even personalize the buttons like making a 'pulse' button which would have a unqiue effect. To use this add pulse class to button:

<a class="btn-floating pulse"><i class="material-icons">favorite</i></a>

Result:

favorite

Grids

The 'Grid' is a grid system that uses columns and row numbers to pinpoint the exact location of items

To use this first use the <div class="row"> which sets up the grid system

To decide what column to place your component in, use <div class="col s1"> where s1 can be changed to s2,s3 etc to decide how many columns your component will span.

A simple example uses the following code where we list icons using the grid system

Example:

First Icon

<div class="row">

<div class="col s4" style="height: 150px">

<i class="material-icons #ef9a9a-text" style="font-size: 150px">event_note</i>

</div>

</div>

Second Icon

<div class="row">

<div class="col s4" style="height: 150px">

<i class="material-icons #ef9a9a-text" style="font-size: 150px">add</i>

</div>

</div>
