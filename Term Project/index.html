Materialize

Popularity 1. Very popular due to its implementation of Google's Material Design which is one of the most favored design languages by developers
2. Gives websites a very consistent and elegant look
Strengths 1. Combines classic principles of good design and essential design principles such as browser portability and responsiveness
2. Extremely light and requires minimal codigng time due to its built-in features including (parallax elements, flow texts, cards, hoverable items and objects)
3. Ease-of-use as it provides detailed documented and specific code examples
4. Easy installation
Weaknesses 1. Anything outside Google's Material Design Framework implementation is possible to become out-of-date
2. Has potential to not work on some browsers other than Google Chrome
3. Large file size makes Materialize a bulky framework to work with
Which developers mostly use it? 1. Front-end developers
2. Web developers who understand basic HTML and CSS
What kind of websites use it? 1. Informative websites (GeeksforGeeks)
2. News websites (La Stampa)
3. Admin websites


Django

Popularity 1. Django has been rapidly gaining popularity for its pragmatic design and ease of use. It also encouranges rapid development and clean design
2. Django is a Python based framework and it is popular because python itself is one of the simplest coding languages
Strengths 1. Django came with documentation far above the usual standard which works well for quick reference
2. Django prevents common security mistakes. Django does it by hiding your site's source code by dynamically generating web pages
3. Django also has a big and interactive community that answer questions fast
Weaknesses 1. Django URL uses regex as URL routing patterns and this makes the code larger and creates complicated syntaxes
2. Django focuses more about dev productivity and backward compatibility than its speed
3. Template errors fail silently by default
Which developers mostly use it? 1. Back-end developers
2. Developers who knows python
What kind of websites use it? 1. Top content-driven websites (The Washington Post, New York Times, The Guardian)
2. Cloud storage platform (Dropbox)
3. Social networks / Platforms (Instagram, Spotify, YouTube)
