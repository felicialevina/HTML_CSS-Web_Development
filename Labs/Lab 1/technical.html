2. How to password protect your page
	I added the script tag to password protect my webpage. 
	In the script tag, I created a pop up window that prompts user to insert the password.
	Then I used a while loop to make sure that if the password does not match, it will prompt the user to enter the password again.
	If it matches, it will go into the webpage.
	

	
3. New tag used and the effect
	The ordered and unordered list tag - it will make a list with numbers for ordered list and bullet points for unordered list
	The image tag - it will import the image into the webpage and in the image tag, we can set the size of the images and arranges the position
	The link tag - this will create links between webpages and we can set it to make it looks like buttons
	The header/paragraph tag - this will produce texts in the webpage and I can choose the fonts, size and spacing of the texts
	

	
4. Does it look different in different browsers and platforms
	Yes. The webpage looks different in different browser and platforms.
	The spacing and arrangement between elements (especially: images) changes when i tried opening it in different browsers (Chrome and Safari) and platforms (laptop and phone).
